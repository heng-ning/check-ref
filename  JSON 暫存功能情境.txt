情境 1：日常使用流程

上傳論文 PDF
系統自動解析並暫存資料
點擊「匯出為 JSON」→ 下載檔案
將 JSON 檔案傳給外部 API 進行驗證
API 回傳驗證結果後，更新 verified_references
================================================================================
情境 2：中斷後恢復

昨天解析了一份文件並匯出 JSON
今天重新開啟網頁（資料已清空）
上傳昨天的 JSON 檔案
資料立即恢復，可以繼續比對
================================================================================
情境 3：批次處理

解析 10 份論文
每份都匯出 JSON
寫一個 Python 腳本批次呼叫 CrossRef API 驗證
將驗證結果寫回 verified_references
================================================================================
